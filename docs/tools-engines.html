Commercial Deployment Options

Deploy as SaaS, private sovereign, multi-tenant, or embedded. Full Docker Compose orchestration with FastAPI, Redis, PostgreSQL, and language runtimes.

🌐 Public SaaS

Best for: General AI research, paper synthesis, public knowledge platform

Infrastructure: GitHub Pages + FastAPI Cloud

Pricing: Free tier, pay-per-API-call for usage

🔒 Private Sovereign

Best for: Enterprise, regulated industries, proprietary research

Infrastructure: Your VPC/servers (Docker Compose)

Pricing: License + support subscription

👥 Multi-Tenant Gateway

Best for: Team collaboration, managed platform service

Infrastructure: Kubernetes + PostgreSQL

Pricing: Per-user/per-org subscription

📦 Embedded SDK

Best for: Integration into existing apps

Infrastructure: Your tech stack (54 packages)

Pricing: One-time SDK license or per-deployment
