## QUICK ACCESS

```javascript
import { CognitiveLanguageRegistry } from './languages/index.js';

// Get all languages
const all = CognitiveLanguageRegistry.getAllLanguages();

// Get language by ID
const cplL = CognitiveLanguageRegistry.getLanguage('cpl-l');

// Parse source code
const ast = cplL.parse(sourceCode);

// Compile to target
const motoko = cplL.compile(ast, 'motoko');
```

## USAGE EXAMPLES

### Parse CPL-L
```javascript
import { CPLLParser } from './languages/cpl-l/src/parser.js';

const parser = new CPLLParser();
const ast = parser.parse(cplLSource);
console.log(ast);
```

### Compile to Motoko
```javascript
import { CPLLCompiler } from './languages/cpl-l/src/compiler.js';

const compiler = new CPLLCompiler();
const motokoCode = compiler.compile(ast);
// Deploy to ICP canister
```

### Execute CPL-P
```javascript
import { CPLPRuntime } from './languages/cpl-p/src/runtime.js';
import { KernelExecutor } from '@medina/organism-runtime-sdk';

const runtime = new CPLPRuntime(new KernelExecutor());
const result = await runtime.execute(cplPCode);
```
